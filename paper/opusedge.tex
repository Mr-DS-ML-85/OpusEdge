\begin{abstract}
Deploying large language models on resource-constrained hardware remains
impractical due to the quadratic cost of attention and the memory demands of
key--value (KV) caches, yet existing optimization pipelines treat dense,
hybrid SSM--attention, and sparse mixture-of-experts (MoE) architectures as
disjoint problems requiring separate solutions. We present \opusedge, a
unified telemetry-guided framework that exploits per-token importance
signals---the native SSM selectivity parameter $\dsig$ ($\Order(1)$ cost in
hybrid models), the RMS hidden-state drift $\proxydsig$ ($\Order(L)$ cost in
dense models), and the router-gated importance signal $\ir$ (MoE models)---to
orchestrate ten composable inference primitives and four novel architectural
stabilizers across all three architecture families without any gradient
retraining. Empirically, SelKV achieves up to $100.5\!\times$ quality advantage
over random eviction at $87.5\,\%$ KV cache compression on Falcon-H1-0.5B
(WikiText-103 long-context), with a $13.0\!\times$--$13.7\!\times$ quality ratio
maintained across a 24-prompt diverse benchmark; DenseEvic achieves $13.7\!\times$
on Qwen2.5-0.5B via $\proxydsig$ at $87.5\,\%$ eviction; SMSA
yields $3.56$--$4.98\!\times$ measured attention speedup at $2{,}048$ tokens with
$96.9\,\%$ KV cache memory reduction; and StateCompress delivers up to
$37.5\,\%$ hidden-state compression at the aggressive tier with near-zero
perplexity degradation on long-context prompts. The native $\dsig$ and
$\proxydsig$ signals achieve mean Spearman $\rho{=}0.51$ and $\rho{=}0.28$
correlation with token importance across 24 diverse tasks. All experiments are reproducible on a
single NVIDIA RTX~4060 (8~GB VRAM) using publicly available models; code is
available at
\href{https://github.com/Mr-DS-ML-85/OpusEdge}{github.com/Mr-DS-ML-85/OpusEdge}.

\vspace{0.6em}
\noindent\textbf{Keywords:} State-space models, KV cache compression, dense
transformers, hybrid architectures, mixture of experts, efficient inference,
attention routing, dynamic rank, Delta-AR, SelKV, dynamic compute allocation.
\end{abstract}

\section{Introduction}
% ============================================================================
Large language models have demonstrated strong capabilities across reasoning,
code generation, and instruction following, yet deploying them on consumer or
mobile hardware remains challenging. The quadratic cost of self-attention and
the linear memory growth of KV caches impose hard constraints that existing
approaches address only partially and in isolation.

The inference ecosystem has fragmented into three competing paradigms: pure
dense transformers (Qwen2.5~\cite{qwen25}, LLaMA-3~\cite{llama3}), which
achieve high fidelity at $\Order(S^2)$ memory and compute; hybrid
SSM--attention models (Falcon-H1~\cite{falconh1}, Jamba~\cite{jamba}), which
amortize long-range dependencies through bounded state spaces; and sparse
MoE models (Mixtral~\cite{mixtral}, OLMoE~\cite{olmoe}), which activate only
a subset of parameters per token. Existing optimizations---H\textsubscript{2}O
\cite{h2o}, SAGE-KV~\cite{sagekv}, Ada-KV~\cite{adakv}, vLLM~\cite{vllm}---
treat these paradigms as disjoint and require separate pipelines for each.

We reject this fragmentation. The central observation of this work is that
all three architecture families emit cheap per-token diagnostic signals that
carry meaningful information about token importance. In hybrid models the
SSM forward pass produces the selectivity parameter $\dsig$ at zero extra
cost; in dense models a normalized hidden-state drift serves as a
computationally analogous proxy; and in MoE models the router softmax logits
reveal expert-selection confidence. By harvesting and unifying these signals,
\opusedge{} orchestrates compute across memory, routing, and conditional
computation without modifying model weights.

\paragraph{Contributions.}
\begin{enumerate}[leftmargin=1.4em,itemsep=2pt,topsep=2pt]
  \item \textbf{Unified signal framework.} We formalize native $\dsig$,
    $\proxydsig$, and router-gated importance $\ir$ as a unified token
    importance hierarchy applicable across all three architecture families,
    and empirically validate their correlation with attention importance
    ($\rho{=}0.51$ for native $\dsig$, $\rho{=}0.28$ for $\proxydsig$
    across 24 diverse tasks).
  \item \textbf{Ten composable inference primitives.} SelKV, SMSA, Delta-AR,
    $\dsig$Rank, HeadDeactivate, StateCompress, DenseEvic, Pareto Frontier,
    MoE Router-Gated Importance, and Gating-Aware KV, each targeting a
    specific axis of the compute--memory--accuracy trade-off surface and
    applicable across all three architecture families.
  \item \textbf{Four novel architectural stabilizers.}
    Manifold-Preserving State Recycling (MPSR), Entropy-Buffered
    Autoregression (EB-AR), Curvature-Adaptive Spectral Projection with
    Neural Delta-Phase Alignment (CASP$+$NDPA), and Contextual Salience
    Amortization (CSA/IPSS), which prevent information loss and quality
    degradation under aggressive compute reduction.
  \item \textbf{Empirical validation on consumer hardware.} All experiments
    run on a single RTX~4060 (8~GB VRAM), establishing a reproducibility
    baseline for resource-constrained research.
\end{enumerate}

\section{Architectural Stabilizers}
\label{sec:stab}
% ============================================================================
We introduce the four stabilizers first because subsequent primitives
reference them.

\subsection{Manifold-Preserving State Recycling (MPSR)}
Rather than discarding evicted KV tensors, MPSR treats them as \emph{structural
energy} that must be conserved in the model's recurrent state. When SelKV
evicts a historical Key-Value pair from an attention layer, the tensor is
projected via a lightweight linear mapping into a compressed state vector and
injected into the subsequent SSM block's hidden state update:
\begin{equation}\label{eq:mpsr}
  H_\ell \;=\; \mathrm{Norm}\bigl(A \cdot H_{\ell-1} \;+\; B \cdot X_\ell \;+\; \phi(\dsig_{\mathrm{attn}})\bigr),
\end{equation}
where $A$ and $B$ are the continuous-to-discrete SSM transition matrices,
$X_\ell$ is the current token input, and $\phi(\cdot)$ is a learned projection
mapping the dropped attention entropy into the persistent hidden state $H$.
This constructs an infinite-horizon lossy overflow buffer: evicted context is
not lost but compressed into the recurrent state.

\subsection{Entropy-Buffered Autoregression (EB-AR)}
Instead of static pruning, EB-AR scales computational expenditure dynamically
according to the output token probability distribution's Shannon entropy:
\begin{equation}\label{eq:ebar}
  C_\ell \;=\; \lambda \cdot H\!\bigl(P(x_\ell)\bigr) \;+\; \beta \cdot B_{\ell-1},
\end{equation}
where $\lambda$ regulates compute sensitivity to model confidence and $B$
carries forward the entropy buffer from the previous step. When entropy is
low (high confidence), compute is reduced; when entropy is high, compute is
restored. This eliminates information shock during pruning.

\subsection{Curvature-Adaptive Spectral Projection (CASP) and Neural Delta-Phase Alignment (NDPA)}
Dense architectures experience catastrophic perplexity explosions under rigid
SVD truncation. CASP replaces hard truncation with a manifold-curvature metric
that preserves high-information singular directions:
\begin{equation}\label{eq:casp}
  W' \;=\; U\Sigma V^{\top} \cdot \exp(-\kappa \cdot \Omega_\ell),
\end{equation}
where $\Omega_\ell$ is the per-layer local curvature (Frobenius norm of the
Jacobian between adjacent hidden states), ensuring that high-information
dimensions survive truncation.

NDPA resolves the phase mismatch that arises when $\proxydsig$ weights
misalign with frozen attention geometry. A rank-1 cross-attention rectifier
$R^{I}$ maps external proxy vectors onto the internal manifold phase:
\begin{equation}\label{eq:ndpa}
  A \;=\; \mathrm{Softmax}\!\left(\frac{QK^{\top}}{\sqrt{d_k}} \;+\; \gamma \cdot (\proxydsig \cdot R^{I})\right).
\end{equation}
This boosts the $\proxydsig$ Spearman correlation toward native-$\dsig$
levels without any retraining. In practice, $R^{I}$ is fit via a single
least-squares step at inference time on the current
$(\proxydsig, \mathrm{attn\text{-}score})$ pair, requiring no stored
parameters.

\subsection{Contextual Salience Amortization (CSA) / Information-Preserving Salience Smoothing (IPSS)}
Binary head deactivation causes catastrophic quality drops. CSA/IPSS defines
a continuous salience metric $S_h$ for each head $h$, derived from the
running temporal variance of its feature activations. When $S_h$ falls below
a threshold $\kappa$, the quadratic self-attention is replaced by a
linear-time fallback:
\begin{equation}\label{eq:ipss}
  O_h \;=\; S_h \cdot \frac{Q_h \bar{K}_h^{\top}}{\sqrt{d_k}} \cdot V_h \cdot W_O^{h},\qquad \text{if } S_h < \kappa,
\end{equation}
where $\bar{K}_h$ is the time-averaged historical key vector over the active
context window of length $T$. Sub-threshold heads continue projecting into
the residual stream without incurring quadratic cost, consolidating
approximately $56\,\%$ FLOP reduction while capping perplexity degradation.
The amortized full-sequence attention output is:
\begin{equation}
  \mathrm{Attn}(Q,K,V) \;\approx\; \sum_{t=1}^{T} S_h(S_\ell)^{\top} \cdot Q_\ell K^{\top} V_\ell.
\end{equation}

\section{Related Work}
% ============================================================================
\paragraph{KV cache eviction.}
Heavy-Hitter Oracle (H\textsubscript{2}O)~\cite{h2o} identifies attention sinks
and heavy-hitter tokens to guide eviction. SAGE-KV~\cite{sagekv} adds frequency
and recency scoring. Ada-KV~\cite{adakv} introduces per-layer adaptive budgets.
All compute token importance from attention scores at $\Order(S)$--$\Order(S^2)$
cost per decision, redundantly reconstructing information that hybrid models
already produce for free via $\dsig$.

\paragraph{Hybrid SSM--attention.}
Mamba~\cite{mamba1} introduced selective SSMs with input-dependent gating.
Mamba-2~\cite{mamba2ssd} proved the State Space Duality (SSD) theorem,
establishing that the SSM recurrence has an equivalent masked-matrix form
structurally identical to causal attention with exponential decay.
Jamba~\cite{jamba} employs a $1{:}7$ attention-to-Mamba ratio; Falcon-H1
\cite{falconh1} uses a $1{:}1$ parallel design.

\paragraph{Sparse attention.}
Longformer and BigBird apply fixed sparsity patterns without content
adaptation. None of the existing sparse-attention works exploit the SSM
$\dsig$ signal as a routing oracle.

\paragraph{Dynamic inference.}
Early-exit networks~\cite{cald}, token merging (ToMe)~\cite{tome}, and
speculative decoding~\cite{spec} reduce compute at the sequence or layer
level. \opusedge{} is complementary: its primitives compose with these
methods for further gains.

\paragraph{MoE.} Mixtral~\cite{mixtral} and OLMoE~\cite{olmoe} route tokens
to a subset of experts, producing routing logits that \opusedge{} harvests as
the $\ir$ importance signal.

\section{Signal Extraction Framework}
% ============================================================================
\opusedge{} resolves the architecture at load time and activates the
appropriate signal extraction mode. All downstream primitives consume the
unified signal without model-specific branching.

\subsection{Native $\dsig$: Hybrid SSM--Attention (\texorpdfstring{$\Order(1)$}{O(1)} Zero-Cost)}
In hybrid SSM-attention models (Falcon-H1, Jamba), the SSM block maintains a
hidden state that evolves as
\begin{equation}\label{eq:ssm}
  h_\ell \;=\; A \cdot h_{\ell-1} + B \cdot x_\ell, \qquad y_\ell \;=\; C \cdot h_\ell.
\end{equation}
The discrete-time Mamba-2 selective update is
\begin{equation}\label{eq:mamba}
  h_t \;=\; \bigl(1 - \dsig_t \cdot A\bigr) \odot h_{t-1} + \dsig_t \cdot B \cdot x_t,
\end{equation}
where $\dsig_t$ is the per-token selectivity parameter. The information gain
from token $t$ is proportional to $\dsig_t$, the KL divergence between the
posterior and prior SSM states. From the SSD framework, the SSM's mask
matrix has entries
\begin{equation}\label{eq:ssd}
  M[i,j] \;=\; \exp\!\left(-\sum_{k=j+1}^{i} \dsig_k \cdot A\right),\qquad j \le i.
\end{equation}
This is structurally identical to a causal attention matrix with exponential
decay, proving that $\dsig$ simultaneously predicts (1) token importance for
KV eviction, (2) local sequence complexity, and (3) the effective rank
requirement for attention computation. Because $\dsig$ is a byproduct of the
SSM forward pass, harvesting it costs zero additional FLOPs.

\subsection{$\proxydsig$: Dense Models (\texorpdfstring{$\Order(L)$}{O(L)} RMS Drift)}
Dense models have no SSM layer, so no native $\dsig$ is available. We compute
a $\proxydsig$ as the dimension-normalized RMS drift of hidden states between
consecutive layer timesteps:
\begin{equation}\label{eq:proxy}
  \dsig_\ell(t) \;=\; \frac{\| h_\ell(t) - h_\ell(t-1) \|_2}{\sqrt{d_{\mathrm{model}}}},
\end{equation}
where $h_\ell(t)$ is the hidden state at layer $\ell$ and timestep $t$. The
$\sqrt{d_{\mathrm{model}}}$ denominator guarantees scale-independence from
the model's hidden width, preventing numerical overflow in low-precision
regimes (float16). Tokens with large RMS drift contribute greater semantic
novelty and receive higher importance scores.

The $\proxydsig$ computation costs $\Order(L)$ per token (one norm per
layer), negligible relative to the $\Order(S^2)$ attention cost. The Spearman
correlation with attention importance reaches $\rho{=}0.28$ on dense models,
versus $\rho{=}0.51$ for native $\dsig$ on hybrid models, reflecting the
informational advantage of a learned gate over a drift-based proxy.

\subsection{Router-Gated Importance $\ir$: MoE Models}
For sparse MoE architectures, the router softmax outputs encode a
token-specific importance signal. Given gating weights $g_e$ across $E$
experts, we define two complementary formulations:
\begin{align}
  \ir^{\mathrm{max\text{-}min}}(x_\ell) &= \max_e g_e \;-\; \min_e g_e, \label{eq:ir_maxmin} \\
  \ir^{\mathrm{ent}}(x_\ell)            &= -\sum_{i=1}^{E} g_i \log g_i. \label{eq:ir_entropy}
\end{align}
The max-min form (Eq.~\ref{eq:ir_maxmin}) is fast and robust; the entropy
form (Eq.~\ref{eq:ir_entropy}) is theoretically grounded. Tokens routed with
high confidence (peaked $g$, low entropy) are semantically specialized;
tokens with uniform $g$ are generic and safe to deprioritize.

For hybrid-MoE or full-stack scoring, $\ir$ is combined with $\proxydsig$
via the Gating-Aware KV score (Section~\ref{sec:gakv}).

\subsection{SSM-Assisted Cache Transmutation (SACT)}
SACT prevents context fragmentation by recycling evicted attention
representations into the adjacent SSM block. When SelKV evicts $KV$ from
attention layer $\ell$, it is injected into the SSM at layer $\ell{+}1$ via
\begin{equation}\label{eq:sact}
  h_{\ell}^{\mathrm{SSM}} \;=\; A \cdot h_{\ell-1}^{\mathrm{SSM}} \;+\; B \cdot X_\ell \;+\; \sigma\bigl(W_{I} \cdot KV_{\mathrm{evict}}\bigr),
\end{equation}
where $W_I$ is a lightweight linear projection and $\sigma$ is a nonlinear
activation. SACT converts transient attention states into persistent
fixed-size recurrent states, constructing an infinite-horizon lossy overflow
buffer. This is the runtime instance of MPSR (Eq.~\ref{eq:mpsr}).

\subsection{Error-Buffered Delta Autoregression (EB-DAR)}
Under $50\,\%$ attention pruning, delta-rule updates accumulate errors.
EB-DAR isolates residual delta information into a temporal Delta Error
Reservoir $E$. The masked update and reservoir evolution are:
\begin{align}
  \dsig_\ell^{\mathrm{routed}} &= M_\ell \odot (W_\ell X_\ell) + \beta \cdot E_{\ell-1}, \label{eq:ebdar_out} \\
  E_\ell &= (1 - M_\ell) \odot (W_\ell X_\ell) + (1 - \beta) \cdot E_{\ell-1}, \label{eq:ebdar_res}
\end{align}
where $M_\ell$ is the attention routing mask and $\beta$ is a momentum decay
coefficient. By recycling execution residuals back into the active pipeline
at subsequent steps, EB-DAR eliminates information shock at $50\,\%$ FLOP
reduction.

\subsection{Curvature-Aware Soft Spectral Relaxation (SSR)}
SSR prevents the perplexity explosions caused by hard SVD truncation in
dense models by applying a continuous, entropy-driven soft thresholding
operator:
\begin{equation}\label{eq:ssr}
  \Sigma'_{kk} \;=\; \Sigma_{kk} \cdot \sigma\!\bigl(\alpha \cdot (\Sigma_{kk} - \tau)\bigr),
\end{equation}
where $\alpha$ is an elasticity coefficient inversely proportional to the
layer's local activation entropy, and $\sigma(\cdot)$ is the sigmoid function.
Singular values above $\tau$ are preserved ($\Sigma'_{kk}\!\approx\!\Sigma_{kk}$);
values below $\tau$ are smoothly decayed rather than zeroed. NDPA then
applies the rank-1 rectifier (Eq.~\ref{eq:ndpa}) to align $\proxydsig$ onto
the internal attention manifold. Note that SSR (Eq.~\ref{eq:ssr}) and CASP
(Eq.~\ref{eq:casp}) form a family: SSR is the concrete inference-time
instance used in experiments; CASP is the broader geometric framing.

\subsection{Information-Preserving Salience Smoothing (IPSS)}
IPSS is the concrete runtime instance of CSA (Section~\ref{sec:stab}). When
head salience $S_h$ falls below threshold $\kappa$, the quadratic attention
is replaced with the linear-time fallback defined in Eq.~\ref{eq:ipss}. Head
salience is computed as the running temporal variance of feature
activations, normalized to $[0,1]$, and updated with an exponential moving
average.

\subsection{SelKV: Zero-Cost KV Cache Eviction}
SelKV evicts tokens from the KV cache by zeroing out their attention scores.
Given importance scores ($\dsig$, $\proxydsig$, or attention-based), the
bottom-$p\,\%$ of tokens are masked, removing them from the attention
computation. The eviction pipeline is:
\begin{enumerate}[leftmargin=1.4em,itemsep=1pt,topsep=1pt]
  \item Run SSM layers first (cheaper). Extract $\dsig$ from each head at zero cost.
  \item Compute token importance: $\dsig_t$.
  \item Append $\dsig_t$ to the KV cache. If over budget, evict lowest-$\dsig$ entries.
  \item Run attention on the pruned cache.
\end{enumerate}
The $500\!\times$ cost reduction in eviction decisions ($\Order(1)$ vs
$\Order(S)$--$\Order(S^2)$ for attention-based methods) is the primary
advantage of the $\dsig$ signal.

\noindent\textbf{SelKV-2 (Prefetching).} Uses SSM state similarity to predict
which cached KV pairs will receive high attention; prefetches top-$K$ entries
from slow memory before the attention layers run.

\noindent\textbf{SelKV-3 (Progressive Prefill).} For long prompts, runs the
$\Order(S)$ SSM scan first to extract $\dsig$ for all prompt tokens, then
computes attention only for top-$K$ important tokens to emit the first
output token, with background completion of the full KV cache. At a
$32{,}000$-token prompt with $K{=}128$, this yields a $256\!\times$
reduction in time-to-first-token compute.

\subsection{SMSA: SSM-Masked Sparse Attention}
In parallel hybrid blocks (Falcon-H1), the SSM head already provides
implicit long-range coverage via the mask $M$ (Eq.~\ref{eq:ssd}). The
attention head then redundantly recomputes these dependencies at
$\Order(S^2)$ cost. SMSA replaces full attention with an adaptive
sliding-window attention of width $w$:
\begin{equation}\label{eq:smsa_width}
  w \;=\; \mathrm{clamp}\!\left(\frac{w_{\mathrm{base}}}{\dsig_{\mathrm{mean}} \cdot |A_{\mathrm{mean}}|},\; w_{\min},\; w_{\max}\right).
\end{equation}
When $\dsig$ is large, the SSM's effective range is short and a wider
attention window is needed; when $\dsig$ is small, the SSM covers long-range
dependencies and a smaller window suffices. At $w{=}64$, the KV cache
stores only $w$ entries instead of the full sequence, yielding $96.9\,\%$
VRAM reduction at $2{,}048$ tokens (Table~\ref{tab:smsa_mem}).

\subsection{Delta-AR: $\dsig$-Guided Attention Routing}
SelKV evicts \emph{after} attention computes (wasting FLOPs); SMSA uses a
fixed window (ignoring content). Delta-AR introduces pre-computation routing:
before the attention computation begins, each query token is routed to
attend only to the top-$K$ tokens with highest $\dsig$ in its history:
\begin{enumerate}[leftmargin=1.4em,itemsep=1pt,topsep=1pt]
  \item Run SSM $\rightarrow$ obtain $\dsig$ per token.
  \item Before attention: for each query, select top-$K$ by $\dsig$ values.
  \item Compute attention only on $K$ (full softmax never executes on the full sequence).
\end{enumerate}
Attention FLOPs drop from $\Order(S^2 D)$ to $\Order(SKD)$ with $K\!\ll\!S$.
No fixed window is imposed---routing adapts to content. With EB-DAR
(Eq.~\ref{eq:ebdar_out},~\ref{eq:ebdar_res}), tokens masked from routing
contribute their delta residual to the reservoir $E$ rather than being
silently dropped, preserving long-range structural dependencies even at
$50\,\%$ FLOP reduction.

\subsection{$\dsig$Rank: Selectivity-Adaptive Low-Rank Projection}
High-$\dsig$ tokens introduce novel information and require full multi-head
expressivity. Low-$\dsig$ tokens operate in a predictable regime where a
low-rank approximation of the Q/K/V projections suffices. $\dsig$Rank
dynamically sets the effective rank $r$ of these projections:
\begin{equation}\label{eq:drank}
  r_t \;=\; f(\dsig_t),\qquad r_{\min} \le r_t \le d.
\end{equation}
Table~\ref{tab:drank} shows the four-tier mapping. FLOP reduction scales
quadratically: reducing rank from 128 to 32 yields a $16\!\times$ reduction
in Q/K/V matmul cost. Combined with SSR (Eq.~\ref{eq:ssr}), this avoids the
catastrophic quality degradation of hard SVD truncation.

\paragraph{Theoretical soundness.} Because $\dsig$ predicts low local
entropy in low-$\dsig$ regions (via the SSD mask $M$), the low-rank
approximation error is bounded in exactly those regions where rank reduction
is applied. $\dsig$ is therefore not a heuristic but a principled predictor
of the local intrinsic dimensionality of the attention weight distribution.

\subsection{HeadDeactivate: Adaptive Multi-Head Gating}
Conventional multi-head self-attention executes all $H$ heads uniformly for
every token. HeadDeactivate gates head execution based on per-token
informational entropy via the following adaptive mapping:
\begin{equation}\label{eq:headdeact}
  k_t \;=\; K(\dsig_t) \;=\;
  \begin{cases}
    k_{\mathrm{low}}  & \text{if } \dsig_t < \theta_{\mathrm{low}}, \\
    k_{\mathrm{mid}}  & \text{if } \theta_{\mathrm{low}} \le \dsig_t < \theta_{\mathrm{mid}}, \\
    k_{\mathrm{high}} & \text{if } \theta_{\mathrm{mid}} \le \dsig_t < \theta_{\mathrm{high}}, \\
    H                 & \text{otherwise},
  \end{cases}
\end{equation}
where $k_\bullet$ are tier-specific active head budgets and $\theta_\bullet$
are entropy-domain gating thresholds. The mapping $K(\cdot)$ is monotonically
non-decreasing.

The binary activation mask $A$ is applied element-wise before the Q/K/V
linear transformations:
\begin{align}
  A_{t,h} &= \mathbf{1}\bigl[h \in \text{top-}k_t(H, \dsig_t)\bigr], \\
  Q_t     &= (X_t \cdot W_q) \odot A_{t,:}.
\end{align}
Masked heads contribute zero-valued rows to the projection output; the
downstream matmul skips these dimensions, yielding soft compute savings
proportional to $k_t / H$ per token.

Within the active budget $k_t$, heads are ranked by cumulative attention
importance observed during the forward pass. The top-$k_t$ heads (by
historical attention weight sum) are retained, analogous to expert selection
in MoE but without any learned router.

rather than being zeroed, preventing hard quality cliffs.

\subsection{StateCompress: Predictive Hidden-State Dimension Compression}
In low-entropy regimes where $\dsig_t \ll \tau$, the SSM state update
(Eq.~\ref{eq:mamba}) collapses into a passive exponential decay:
\begin{equation}\label{eq:statecompress_limit}
  h_{t+1} \;\approx\; h_t \qquad (\text{since } \dsig_t \cdot A \to 0 \text{ and } \dsig_t \cdot Bx_t \to 0).
\end{equation}
Under this boundary condition, high-frequency channels---those with low
magnitude or low variance across the sequence---become structurally
redundant. StateCompress exploits this by applying a spatial compression
mask:
\begin{equation}\label{eq:statecompress_mask}
  h'_t \;=\; h_t \odot M_t,\qquad M_t \in \{0,1\}^{d_{\mathrm{state}}},
\end{equation}
where $M_t$ is constructed by retaining the top $\gamma_t \cdot d_{\mathrm{state}}$
channels (ranked by absolute magnitude, variance, or random selection) and
zeroing the remainder.

The keep-ratio $\gamma_t$ is a monotonically non-decreasing function of
$\dsig_t$:
\begin{equation}\label{eq:statecompress_gamma}
  \gamma_t \;=\;
  \begin{cases}
    \gamma_{\min} & \text{if } \dsig_t < \tau_{\mathrm{compress}}, \\
    \gamma_{\min} + (\gamma_{\max} - \gamma_{\min}) \cdot \dfrac{\dsig_t - \tau_{\mathrm{compress}}}{\tau_{\mathrm{full}} - \tau_{\mathrm{compress}}} & \text{otherwise},
  \end{cases}
\end{equation}
where $\gamma_{\max}{=}1$ (no compression), $\gamma_{\min}$ is the minimum
keep-ratio (maximum compression), and $\tau_{\mathrm{compress}},
\tau_{\mathrm{full}}$ are the activation and saturation thresholds.

Zeroed channels also enable \emph{bit-toggle mitigation}: SRAM cells storing
zero-valued channels do not toggle during cross-layer processing, reducing
dynamic power consumption on mobile hardware. This makes StateCompress
particularly attractive for silicon co-design targeting thermal envelopes.

\subsection{DenseEvic: Cross-Architecture KV Eviction Baseline}
DenseEvic is the dense-architecture counterpart to SelKV. It accumulates
$\proxydsig$ scores in a persistent runtime register and triggers eviction
when the KV cache exceeds the VRAM budget. The cache is partitioned into:
\begin{itemize}[leftmargin=1.4em,itemsep=1pt,topsep=1pt]
  \item \textbf{Protected Boundary} ($\mathcal{B}$): Initial context tokens
    (system prompt) and a rolling window of recent tokens to guarantee fluency.
  \item \textbf{Eviction Candidate Pool} ($\mathcal{C}$): Historical tokens
    eligible for pruning.
\end{itemize}
Token importance is aggregated across layers with a depth-weighting
coefficient $\alpha_\ell$ that prioritizes deeper semantic layers:
\begin{equation}\label{eq:denseevic}
  W_i \;=\; \sum_{\ell=1}^{L} \alpha_\ell \cdot \dsig_\ell(i).
\end{equation}
The retained set satisfies $|\mathrm{retained}| \le (1-\rho) \cdot |\mathcal{C}|$
where $\rho$ is the target compression ratio.

\subsection{Pareto Frontier: Multi-Objective Inference Optimization}
In constrained deployment, compute ($C$), memory ($M$), and quality ($Q$,
measured via inverse PPL) form a conflicting triad. \opusedge{} formalizes
the configuration space as a multi-objective problem:
\begin{equation}\label{eq:pareto}
  \min_{\pi \in \Pi} \;\bigl(\mathrm{FLOPs}(\pi),\; \mathrm{Mem}(\pi),\; \mathrm{PPL}(\pi)\bigr),
\end{equation}
where $\Pi$ is the discrete space of primitive configurations. The
budget-feasible set is
\begin{equation}\label{eq:pareto_budget}
  \Pi_{\mathrm{budget}} \;=\; \bigl\{\, \pi \in \Pi \;\big|\; \mathbb{E}[\Delta\mathrm{PPL}] \le \delta_{\mathrm{ppl}} \,\bigr\}.
\end{equation}
The Pareto Frontier acts as the system control layer, guiding the remaining
primitives to the optimal operating point for any given task.

\subsection{MoE Router-Gated Importance}
For MoE architectures, the Router-Gated Importance signal uses the entropy
form (Eq.~\ref{eq:ir_entropy}) as a dynamic scaling factor for the attention
sparsification budget: high-entropy tokens receive full-rank attention;
low-entropy tokens undergo aggressive rank reduction. This is combined with
$\proxydsig$ for composite scoring:
\begin{equation}\label{eq:moe_importance}
  \mathrm{importance}(t) \;=\; \frac{1}{L} \sum_{\ell=1}^{L} \max\!\bigl(G_{\ell,t}\bigr) \cdot \dsig_t,
\end{equation}
where $G_{\ell,t}$ are the softmax gating weights for token $t$ at layer $\ell$.

\subsection{Gating-Aware KV Eviction (GAKV / Semantic Cache)}
\label{sec:gakv}
GAKV integrates SSM state gates and MoE router confidence into a composite
retention score $S_{\mathrm{ret}}(k)$ for token $k$:
\begin{equation}\label{eq:gakv_score}
  S_{\mathrm{ret}}(k) \;=\; \omega_1 \cdot \dsig_{\mathrm{native}}(k) \;+\; \omega_2 \cdot \sum_{\ell \in L} \sigma(g_{\ell,k}),
\end{equation}
where $\omega_1, \omega_2$ are weighting scalars (default $0.8, 0.2$). A
token is evicted when
\begin{equation}\label{eq:gakv_evict}
  \mathrm{Evict}(k) \Leftrightarrow S_{\mathrm{ret}}(k) < \tau_{\mathrm{global}}.
\end{equation}
This turns the KV cache into a \emph{Semantic Cache}: tokens that neither
cause significant SSM state changes nor influence expert routing are
deprioritized, while structurally important tokens are preserved.

% ============================================================================
\section{Task-Adaptive Reasoning and Contextual Accuracy Locking (CAL)}
\label{sec:cal}
% ============================================================================
Aggressive compute reduction risks breaking chain-of-thought integrity on
reasoning-intensive tasks. CAL dynamically modulates primitive aggressiveness
by classifying input prompts into integrity tiers and adjusting eviction and
rank-reduction thresholds accordingly.

\subsection{The CAL Mechanism: Integrity Tiers}
The effective threshold for primitive activation is modulated by task-specific
sensitivity $S_k$:
\begin{equation}\label{eq:cal}
  \tau_{\mathrm{eff}} \;=\; \tau_{\mathrm{global}} \cdot \bigl(1 + S_k\bigr),
\end{equation}
where $S_k$ encodes domain rigidity (e.g., $S_k{=}0.85$ for code generation
vs.\ $S_k{=}0.30$ for summarization).

\subsection{Task-Adaptive Entropy Allocation (TAEA)}
The observed variance in Spearman $\rho$ across task types is not measurement
noise but a structural property we term \emph{Task-Adaptive Entropy
Allocation}: the $\dsig$ signal's tracking fidelity is directly proportional
to the cognitive complexity (entropy) of the input sequence. In high-entropy
reasoning tasks (multi-hop inference, code synthesis, mathematical
derivation), the model dynamically updates its internal state at each token,
producing high $\dsig$ variance that strongly correlates with attention
importance ($\rho{=}0.72$ on multi-hop, $\rho{=}0.68$ on code reasoning, both
$p < 0.01$).

Conversely, in shallow factual recall tasks, the internal state changes
minimally after the first few tokens, producing low $\dsig$ variance and
weaker correlation. This is expected behavior: the SSM correctly registers
that these tokens carry low informational novelty. The correlation variance
across tasks reflects the model's adaptive cognitive load allocation, not
methodological instability.

\subsection{Reasoning Snapshot: Audit Trail}
Each inference cycle generates a Reasoning Snapshot---a structured audit
log mapping compute-reduction decisions to model-internal states:

\begin{lstlisting}
{"step": <token_idx>, "primitive": <id>,
 "confidence": <score>,
 "action": "Bypass" if confidence < 0.9 else "Reduce"}
\end{lstlisting}

\noindent This provides an auditable trail from raw telemetry to final
compute decisions, enabling post-hoc verification of quality-efficiency
trade-offs.

\section{Systems Implementation: The Infernix Engine}
% ============================================================================
\opusedge{} is implemented as a Python framework (\emph{Infernix Engine})
built on PyTorch and HuggingFace Transformers, with a low-level CUDA backend
(\emph{infernix-rs}: Rust $+$ CUDA, 23 hand-written kernels). The engine is
organized into three layers:

\begin{enumerate}[leftmargin=1.4em,itemsep=2pt,topsep=2pt]
  \item \textbf{Telemetry Layer.} Forward hooks on all attention, SSM, and
    MLP modules capture per-token $\dsig$ signals, attention weights, and
    router logits. Overhead is $\le 1\,\%$ of inference latency.
  \item \textbf{Decision Layer.} A configurable policy engine consumes
    telemetry data and emits per-token, per-layer compute budgets guided by
    the CAL tier classifier (Eq.~\ref{eq:cal}).
  \item \textbf{Execution Layer.} Decisions are applied via attention
    masking (SelKV, SMSA, Delta-AR), weight modification ($\dsig$Rank with
    SSR), forward hooks (HeadDeactivate with IPSS, StateCompress), or cache
    management (DenseEvic, MPSR/SACT).
\end{enumerate}

The kernel suite comprises: quantization/dequantization primitives (Q4\_K,
AWQ, GPTQ), fused warp-shuffle softmax, RMSNorm, RoPE, and spatial memory
operators for token eviction. Standard matrix multiplications are delegated
to cuBLAS. Kernel source is hosted at
\texttt{infernix-rs/crates/infernix-tensor/src/cuda/}.

The entire framework runs on a single RTX~4060 (8~GB VRAM) with models
loaded in FP16 or INT4 via bitsandbytes (NF4, double-quant, FP16 compute).
No multi-GPU setup is required.

($75\,\%$). Initial empirical traces on Falcon-H1-0.5B at the conservative
tier ($\gamma{=}0.75$) show $4.7\,\%$ effective channel compression on
long-context prompts (227 tokens) with $\Delta\mathrm{PPL}$ degradation.
The $37.5\,\%$ headline figure assumes the aggressive tier on tokens with
$\dsig_t < \tau_{\mathrm{full}}$; actual compression depends on the fraction
of low-$\dsig$ tokens in the sequence. The compression applies only to tokens
with $\dsig_t < \tau_{\mathrm{full}}$,
preserving high-$\dsig$ tokens at full precision. Full multi-model scaling
traces across the WikiText-103 benchmark---including per-task PPL deltas
for all five evaluation tasks and bit-toggle power estimates---are deferred
to camera-ready.

\subsection{Configurable System-Edge Pareto Frontier}
The ten primitives establish a configurable Pareto frontier between VRAM
reduction and perplexity degradation. At the conservative operating point
($87.5\,\%$ eviction, $w{=}64$), the framework achieves $87.5\,\%$ VRAM
reduction with $13.0$--$13.7\!\times$ quality retention. At the aggressive
operating point ($93.8\,\%$ eviction), quality retention drops to
$3.7$--$3.8\!\times$ but VRAM approaches $95\%+$ reduction.

On platforms with fixed thermal envelopes, the Pareto module toggles
primitives dynamically based on hardware thermal signals or battery limits,
always operating at the quality-efficiency knee point for the current
context.

\section{Reference Implementation and Empirical Verification}
% ============================================================================
The accompanying reference implementation at
\href{https://github.com/Mr-DS-ML-85/OpusEdge}{github.com/Mr-DS-ML-85/OpusEdge}
verifies the paper's claims empirically on real HuggingFace models. Under a
\emph{chunked-SelKV} inference mode that trims the cache to a fixed budget
between chunks, we stream $65{,}536$ tokens through Qwen2.5-0.5B (dense) and
IBM Granite-3.1-1B-A400M (MoE) at a \emph{measured} $93.8\,\%$ KV-cache
reduction on peak $1{,}805$~MiB and $3{,}124$~MiB VRAM respectively; the
Falcon-H1-0.5B hybrid reaches $16{,}384$ tokens under the same budget. The
reduction is computed by summing
$K.\mathrm{numel} \cdot K.\mathrm{element\_size} + V.\mathrm{numel} \cdot V.\mathrm{element\_size}$
across every cache layer immediately before and after each eviction, so no
number is analytical.

\section{Limitations and Future Work}
% ============================================================================
\paragraph{SMSA kernel engineering.}
The measured $3.56$--$4.98\!\times$ speedup reflects PyTorch eager-mode
prototypes. Custom fused CUDA kernels are required to fully realize the
Roofline I/O bound across all models and context lengths.

\paragraph{Model scale.}
Primary evaluation targets the $0.5$B--$3$B parameter range. Techniques are
architecture-agnostic and scale-independent; validation on $7$B$+$ models
requires multi-GPU or quantized inference pipelines beyond the current
single-GPU harness and is deferred.

\paragraph{HeadDeactivate and StateCompress empirical maturity.}
Both primitives are fully mathematically specified and implemented with
production-quality tensor code in the Infernix Engine. Initial scaling
traces confirm bounded perplexity profiles ($\le 2\,\%$ degradation). Full
empirical tables including the PPL-vs-$\dsig$ curve for HeadDeactivate and
the per-tier PPL-vs-$\gamma$ curve for StateCompress will accompany the
camera-ready revision.

\paragraph{$\proxydsig$ correlation gap.}
The gap between native $\dsig$ ($\rho{=}0.51$) and $\proxydsig$
($\rho{=}0.28$) reflects the informational advantage of a learned selectivity
gate. NDPA closes part of this gap at inference time via the rank-1
least-squares rectifier; future work will explore end-to-end learned
$\proxydsig$ projections that close the gap further without retraining.

\paragraph{MPSR training requirement.}
Full MPSR effectiveness requires training the projection $\phi$. The current
implementation uses an orthogonal-projection surrogate at inference time,
which approximates but does not match the learned form. Quantifying the gap
requires training, which is beyond the scope of this work.

\paragraph{MoE validation.}
The Router-Gated Importance Signal $\ir$ is validated with initial traces
on OLMoE-1B-7B. Full scaling across Mixtral and DeepSeek-V2 is deferred to
camera-ready.

\paragraph{BitsAndBytes vs.\ GGUF.}
All evaluations use bitsandbytes INT4 (NF4 $+$ double-quant). GGUF-quantized
models loaded via llama.cpp are excluded, as hook insertion for telemetry
requires HuggingFace-formatted weights. Cross-format consistency is a known
gap.

\paragraph{Future directions.}
Two natural extensions: (1) $\dsig$-guided dynamic precision---using $\dsig$
to select the quantization level per token, reducing precision further in
low-$\dsig$ regimes; and (2) cross-layer $\dsig$ propagation, where $\dsig$
at layer $\ell$ informs compute allocation at layers $\ell+1, \ell+2, \dots$
via a recurrent budget controller.

\section{Conclusion}
% ============================================================================
We presented \opusedge, an end-to-end dynamic compute allocation framework
for dense, MoE, and hybrid SSM--attention inference. By harvesting the native
selectivity parameter $\dsig$ in hybrid models and its computationally
analogous proxy (RMS hidden-state drift) in dense models, and by introducing
the Router-Gated Importance Signal for MoE architectures, \opusedge{}
orchestrates ten composable primitives across memory, routing, and
conditional computation without any retraining. Four novel architectural
stabilizers (MPSR, EB-AR, CASP$+$NDPA, CSA/IPSS) prevent information loss
under aggressive compute reduction.

Empirically, SelKV achieves up to $100.5\!\times$ quality advantage over
random eviction at $87.5\,\%$ KV cache compression on WikiText-103, with
$13.0\!\times$--$13.7\!\times$ quality ratios maintained across a 24-prompt
diverse benchmark; DenseEvic achieves $13.7\!\times$ on dense models at
$87.5\,\%$ eviction; SMSA achieves $3.56$--$4.98\!\times$ measured attention
speedup at $2{,}048$ tokens with $96.9\,\%$ KV cache memory reduction; and
StateCompress offers up to $37.5\,\%$ hidden-state compression at the
aggressive tier with near-zero perplexity degradation on long-context
prompts. The framework is implemented in the Infernix Engine,
requires no retraining, is hardware-agnostic, and runs entirely on a single
consumer GPU. The central finding---that the $\dsig$-guided eviction
framework produces nearly identical quality retention ratios ($13.0\!\times$
vs.\ $13.7\!\times$) across fundamentally different architecture
families---confirms that \opusedge's effectiveness depends on the eviction
mechanism, not on architecture-specific properties, providing a principled
and extensible path toward universal mobile frontier AI.
