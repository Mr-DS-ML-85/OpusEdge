OpusEdge is a telemetry-guided compute-allocation framework for dense, MoE, and hybrid SSM–attention LLMs. A single per-token importance signal — Δ — drives 10 core primitives, 4 stabilizers, and 2 task controllers across KV eviction, sparse attention, head gating, state compression, and low-rank projection. No retraining. Runs on a single RTX 4060.

Scene 01 · SelKV eviction

Evict what the model already knows is boring.

Every KV entry in the cache has a Δ score attached — free in hybrid models, O(L) in dense ones. SelKV drops the bottom-p% before the attention kernel runs. At 87.5% compression, PPL degrades by 5.16 — random eviction blows up to 519.01. That's 100.5×.

// drop the bottom 87.5% auto r = SelKV::evict(delta, 0.875, delta.size()); // r.retained_indices ← 12.5% of the cache survives // r.evicted_indices ← everything else is gone
Scene 02 · SMSA sliding window

Don't recompute what the SSM already covered.

In parallel hybrid blocks the SSM head already carries long-range dependencies via the SSD mask. Attention only needs a local window — SMSA makes that window adaptive, scaled by 1/Δ.

At w = 64 and 2,048 tokens: 3.56–4.98× measured speedup, 96.9% VRAM reduction. PyTorch eager mode. No custom kernels needed.

Scene 03 · Delta-AR routing

Route before the softmax runs.

SelKV evicts after attention wastes FLOPs. SMSA uses a fixed window. Delta-AR is the third option — before softmax runs, each query picks its top-K key positions by Δ. Attention drops from O(S²) to O(S·K).

Masked tokens don't vanish — they feed the EB-DAR reservoir, and re-enter when their residual grows past μ + 0.1σ. No information shock at 50% pruning.

Scene 04 · StateCompress channels

Zero the channels that stopped mattering.

When Δ ≪ τ the SSM state update collapses to passive decay. High-frequency channels become structurally redundant. StateCompress zeros them by magnitude — 37.5% hidden-state reduction at near-zero PPL cost.

Bonus on mobile silicon: zeroed SRAM cells don't toggle across layers, cutting dynamic power on the chip's thermal envelope.

Scene 05 · full primitive orbit

10 core + 4 stabilizers + 2 controllers.

Every primitive is a pure function of Δ. Compose freely; stabilizers prevent the cliffs; the CAL family modulates aggressiveness by task.

Core · 10

KV eviction

SelKV

Drop bottom-p% tokens by Δ before attention runs. 100.5× vs random at 87.5%.

Sparse attention

SMSA

Adaptive sliding window scaled by 1/Δ. 4.98× at 2,048 tokens.

Sparse attention

Delta-AR

Per-query top-K routing before softmax. O(S²) → O(S·K).

Rank

ΔRank

Per-token low-rank Q/K/V (16/32/64/128 tiers). 64× on Q/K/V matmul.

Heads

HeadDeactivate

Gate heads by token entropy. 87.5% off on function-word tokens.

State

StateCompress

Zero low-magnitude channels when Δ ≪ τ. 37.5% state at ~0 PPL.

KV eviction

DenseEvic

Protected-boundary + candidate-pool eviction for dense models. 13.7×.

Composite

GAKV / R-GAKV

Semantic cache — evicts only when Δ and IR both agree.

Search

Pareto Frontier

Sweep (ratio, window, rank, keep) → knee-point selector.

MoE

Router-IR

Per-token routing confidence (max-min or entropy). Feeds GAKV.

Stabilizers · 4

state recycle

MPSR / SACT

Project evicted KV into the SSM state so no context is lost — infinite-horizon overflow buffer.

compute buffer

EB-AR / EB-DAR

Scale compute by output entropy; recycle masked delta residuals through a temporal reservoir.

soft SVD

SSR / CASP + NDPA

Sigmoid-gated soft rank truncation + rank-1 phase rectifier for Proxy-Δ.

head fallback

IPSS / CSA

Sub-salient heads fall through to a linear-time K̄ path instead of hard-zeroing.

Task controllers · 2

rigidity map

CAL

Classify the prompt into a rigidity tier (code/math/summarize/…) and modulate every threshold by (1 + rigidity).

confidence freeze

R-CAL

EMA over token log-prob confidence; freeze recomputation once the model has settled on a stable regime.

Under the hood: 16 C++ headers, ~50 member functions, and 30 Python bindings (see the all_primitives_demo output).
